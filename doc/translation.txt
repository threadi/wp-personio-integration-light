Translations
=============

* Plugin detect language of WordPress and set this as main language on installation and for the first import
* user can choose another language for import and as main language

Hints
-----
* Personio sends all taxonomy-data in english no matter which language is used for XML-request
* some values are fixed in Personio and therefore predefined
-> these are translated by the plugin
* other values can be changed by the user in personio
-> the user can change these values in WordPress afterwards
